# Подсказка по GIT

Создание репозитория
```sh
git init
```

Отслеживание файла
```sh
git add
```

Коментарий к добавленному-измененному файлу
```sh
git commit -m "Message"
```

Отслеживание Изменений
```sg
git log
```

Краткий список изменений
```sh
git log --oneline
```

Переход между изменениями
```sh
git checkout
```
